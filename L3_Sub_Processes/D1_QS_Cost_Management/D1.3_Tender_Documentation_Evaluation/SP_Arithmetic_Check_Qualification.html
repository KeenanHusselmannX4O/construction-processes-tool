D1.3 — Sub-Process

Arithmetic Check & Qualification Review

L1: QS & Cost Management → L2: Tender Documentation & Evaluation → L3: Arithmetic Check & Qualification
Process ID: D1.3-SP-002Value Chain: Phase 3Version 2.0April 2026
↑ L2 D1.3 Tender Docs ← Tender Doc Compilation ● SP: Arithmetic Check → Comparative Analysis ↓ L4 Tasks

1. Purpose

Verify the arithmetic accuracy of all received tender submissions and assess tenderer qualification (CIDB grading, tax compliance, B-BBEE, returnable schedules). This is the first gate in the evaluation process — tenders with material arithmetic errors or qualification failures are handled per the tender instructions before proceeding to comparative analysis.

Trigger

Tender closing date reached; sealed tenders received and opened in accordance with tender procedures

Accuracy

100% arithmetic verification; all errors catalogued per item with corrected totals calculated

Output

Arithmetic check schedule; qualification compliance matrix; corrected tender totals for ranking
