Feasibility Cost Estimate

L1: QS & Cost Management → L2: Cost Planning & Estimation → L3: Feasibility Cost Estimate
Process ID: D1.1-SP-001 Value Chain: Phase 1 (Feasibility & Investment) Version 2.0 April 2026
↑ L2 D1.1 Cost Planning ● SP: Feasibility Estimate → Elemental Cost Plan ↓ L4 Tasks

1. Purpose

Produce an order-of-magnitude cost estimate to determine financial viability of a proposed development. This is the earliest QS involvement in the project lifecycle, typically performed from concept parameters before detailed design exists.

Trigger
Developer requests feasibility assessment for proposed project
Accuracy Range
±15–25% (ASAQS Class 4/5 estimate)
Output
Feasibility cost report with budget recommendation
