1. Task Overview

This task set expands the Variation Registration & Cost Assessment sub-process into granular work instructions. Each task ensures rigorous management of variation orders from identification through to formal approval: registering the variation with a unique VO number, classifying the type and selecting the valuation method, requesting and assessing contractor quotations, conducting independent QS cost assessments, negotiating agreed values, obtaining employer approval, and issuing the formal variation order. Disciplined variation management per JBCC clauses 17–19 and NEC4 clauses 60–65 protects both employer and contractor interests.

2. Detailed Task Breakdown

TASK 1.6.1-01

Register Variation with Unique VO Number & Source Documentation

Create a formal entry in the variation register the moment a potential variation is identified. Assign a unique sequential VO number, record the source (PA instruction, design revision, site condition, employer request), and link to the originating documentation.

Role
Quantity Surveyor
SLA
1 working day from identification
Inputs
PA instruction; revised drawing; contractor notice; employer change request
Outputs
Variation register entry with VO number; source document filed and cross-referenced
Tools
Variation register template; document management system; platform VO module
Standard
JBCC clause 17 (PA Instructions); NEC4 clause 61 (Compensation Events notification)
  • Work Instruction: Upon receiving a PA instruction, revised drawing, or contractor notice, immediately open the variation register
  • Assign the next sequential VO number using the project numbering convention (e.g., VO-001, VO-002)
  • Record: VO number, date identified, source (PA instruction / design revision / site condition / employer request), brief description of the scope change
  • File the originating document (instruction letter, drawing revision, site memo) and link to the register entry
  • Set the initial status to "Registered — Pending Assessment" and notify the project team of the new variation
  • ✓ Quality Check: VO number unique and sequential; source document filed; all register fields populated; team notified
TASK 1.6.1-02

Classify Variation Type & Select Valuation Method

Classify the registered variation by type (scope change, site condition, design revision) and determine the appropriate valuation method based on whether comparable BoQ rates exist. The classification drives the valuation approach and the contractor's entitlements.

Role
Quantity Surveyor
SLA
0.5 working day
Inputs
VO register entry; variation scope description; priced BoQ; contract conditions
Outputs
Classification recorded; valuation method selected and documented
Tools
Variation classification matrix; BoQ rate reference; valuation method guide
Standard
JBCC clause 18 (Adjustment types); NEC4 clause 63 (Assessing compensation events)
  • Work Instruction: Review the variation scope and classify: (a) Scope change by employer, (b) Unforeseen site condition, (c) Design revision by architect/engineer, (d) Regulatory requirement
  • Check the BoQ for comparable work items — does identical or similar work exist with contract rates?
  • Select the valuation method: Contract Rates (identical work in BoQ), Analogous Rates (similar but adjusted), Star Rates (first principles build-up), Day Works (cannot predetermine scope), or Lump Sum Quotation (clearly defined additional scope)
  • Document the rationale for the selected method in the variation register — this is auditable and may be challenged
  • Update the register status to "Classified — Awaiting Quotation/Assessment"
  • ✓ Quality Check: Classification aligns with contract definitions; valuation method is justifiable; rationale documented
TASK 1.6.1-03

Request Contractor Quotation with Supporting Breakdown

Formally request a detailed quotation from the contractor for the cost impact of the variation. The request must clearly describe the scope and require the contractor to provide a full cost breakdown (labour, materials, plant, subcontractors, overheads, profit) to enable meaningful comparison.

Role
Quantity Surveyor
SLA
Issue request immediately; contractor response 7–14 days
Inputs
VO scope description; relevant drawings; specification amendments
Outputs
Formal quotation request issued; contractor quotation received with breakdown
Tools
Quotation request template; project correspondence system
Standard
JBCC clause 29 (Adjustment of Contract Value); contract-specific quotation requirements
  • Work Instruction: Prepare a formal quotation request letter referencing the VO number, description, and scope of the variation
  • Attach all relevant documents: revised drawings, specification amendments, site photographs, PA instruction
  • Specify the required breakdown format: quantities, labour rates, material costs, plant rates, subcontractor quotes, OH&P percentages
  • Set a response deadline (typically 7–14 working days depending on complexity) and note it in the register
  • Issue the request formally via the project correspondence system with read-receipt or delivery confirmation
  • Track the response deadline — follow up if the contractor does not respond within the stipulated period
  • ✓ Quality Check: Request references correct VO number; scope clearly described; all attachments included; deadline set and tracked
TASK 1.6.1-04

Prepare Independent QS Cost Assessment

Independently assess the cost impact of the variation using the valuation method selected in Task 1.6.1-02. This assessment is prepared without reference to the contractor's quotation to ensure objectivity. Build rates from contract data, market rates, or first principles as appropriate.

Role
Quantity Surveyor
SLA
2–5 working days (complexity-dependent)
Inputs
VO scope; contract BoQ rates; market rate database; first-principles cost data; Stats SA CPA indices
Outputs
QS independent cost assessment with full breakdown and methodology
Tools
Rate build-up template; cost database; BoQ pricing software; BER cost data
Standard
SACQSP professional practice; ASAQS measurement rules; JBCC clause 29 valuation principles
  • Work Instruction: If Contract Rates method: extract the applicable BoQ rate and apply to the measured quantity of varied work
  • If Analogous Rates: adjust the comparable BoQ rate proportionally — document the adjustment logic (e.g., higher specification = rate + 15%)
  • If Star Rates: build the rate from first principles — labour (man-hours × wage rate), materials (quantities × supplier prices), plant (hours × hire rate), subcontractor quotes, OH&P margin per contract
  • If Day Works: estimate the expected labour hours, material quantities, and plant usage at the day work rates in the BoQ
  • Present the assessment in the same breakdown format requested from the contractor for direct comparison
  • File the assessment separately from the contractor quotation to maintain audit independence
  • ✓ Quality Check: Methodology aligns with selected valuation method; rates sourced and referenced; assessment independently prepared
TASK 1.6.1-05

Negotiate & Agree Variation Value with Contractor

Compare the QS independent assessment against the contractor's quotation. Where the variance is within 10%, propose agreement on a mid-point. Where the variance exceeds 10%, conduct a line-by-line negotiation using contract rates as the anchor. The agreed value is binding for payment purposes.

Role
QS + Contractor QS
SLA
3–10 working days (negotiation period)
Inputs
QS assessment (Task 1.6.1-04); contractor quotation (Task 1.6.1-03)
Outputs
Agreed VO value; signed negotiation record; updated register
Tools
Comparison schedule; negotiation minutes template
Standard
JBCC clause 29; fair dealing principles; SACQSP professional conduct
  • Work Instruction: Prepare a side-by-side comparison: QS assessment vs contractor quotation — by line item and total
  • Calculate the overall variance: (contractor quote − QS assessment) ÷ QS assessment × 100 = variance %
  • If variance < 10%: propose agreement at the mid-point or QS-assessed value with minor adjustment
  • If variance > 10%: conduct a line-by-line negotiation — identify which items drive the variance and negotiate each separately
  • Use contract rates as the primary anchor — the contractor cannot claim rates significantly above BoQ rates for similar work
  • Record the agreed value in the negotiation minutes — both QS parties sign the agreement
  • If agreement cannot be reached, refer to the PA for determination per the contract dispute resolution clause
  • ✓ Quality Check: Comparison schedule prepared; negotiation documented; agreed value signed by both parties; register updated
TASK 1.6.1-06

Prepare Employer Cost Impact Summary & Obtain Approval

Prepare a cost impact summary for the employer showing the agreed variation value, its impact on the contract sum, effect on contingency, and overall budget implications. The employer must approve the variation expenditure before a formal VO can be issued.

Role
QS → Principal Agent → Employer
SLA
3–5 working days (approval cycle)
Inputs
Agreed VO value; current contract sum; contingency balance; PFC impact
Outputs
Employer cost impact report; employer approval or rejection; approval on file
Tools
Cost impact report template; PFC model; contingency tracker
Standard
JBCC employer approval requirements; project-specific delegation of authority
  • Work Instruction: Prepare the cost impact report showing: VO description, agreed value, source of funds (contingency / additional budget)
  • Show the impact on the contract sum: current contract sum + this VO = new adjusted contract sum
  • Show the impact on contingency: current contingency balance − this VO (if from contingency) = remaining contingency
  • Update the PFC to reflect the variation and present the revised projected final cost to the employer
  • Route through the PA for recommendation, then to the employer for formal approval
  • If the employer rejects: record the rejection, advise on implications (contractor may have a contractual entitlement), and escalate if necessary
  • ✓ Quality Check: Impact report shows full budget implications; PA recommendation included; employer approval formally documented
TASK 1.6.1-07

Issue Variation Order & Update Contract Sum

Upon employer approval, issue the formal Variation Order (VO) document to the contractor. Update the variation register status to "Approved & Issued", adjust the contract sum, and flag the VO for inclusion in the next interim payment certificate (D1.5-SP-002).

Role
Quantity Surveyor
SLA
1 working day from employer approval
Inputs
Employer approval; agreed VO value; VO document template
Outputs
Formal VO issued; variation register updated; contract sum adjusted; flagged for next IPC
Tools
VO template; variation register; contract sum adjustment schedule; platform finance module
Standard
JBCC clause 29 (formal VO requirements); NEC4 clause 65 (implementing compensation events)
  • Work Instruction: Prepare the formal VO document: VO number, description, agreed value, employer approval reference, effective date
  • Issue the VO to the contractor with a cover letter confirming the approved value and instructing execution
  • Update the variation register: change status from "Pending Approval" to "Approved & Issued"; record the issue date and value
  • Adjust the contract sum in the master financial register: original contract sum + cumulative approved VOs = adjusted contract sum
  • Flag the VO for inclusion in the next interim payment certificate — notify the valuation team (D1.5-SP-002)
  • Distribute copies: contractor (original), PA (copy), employer (copy), project file (copy)
  • ✓ Quality Check: VO formally issued; register updated; contract sum adjusted; payment inclusion flagged; all parties have copies
