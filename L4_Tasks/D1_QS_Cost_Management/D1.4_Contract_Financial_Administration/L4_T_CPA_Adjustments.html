1. Task Overview

This task set details the monthly Contract Price Adjustment (CPA) calculation cycle per JBCC CPA provisions. Tasks cover obtaining current BER Building Cost Index data from Stats SA, retrieving the base-month index from the contract price schedule, computing the index movement factor, determining the qualifying value from the current interim certificate, applying the CPA formula (V × (I₁ − I₀) / I₀), preparing the CPA calculation schedule, and recording cumulative CPA in the financial tracking register. This recurring cycle runs with every monthly valuation.

2. Detailed Task Breakdown

TASK 1.4.2-01

Obtain Current BER Building Cost Index

Source the latest published BER Building Cost Index for the certification month. The BER index is published monthly by the Bureau of Economic Research at Stellenbosch University and is the primary index used in JBCC CPA calculations. Also retrieve any supplementary Stats SA PPI indices if the contract specifies a weighted formula.

Role
Quantity Surveyor
SLA
On publication date (typically 15th of month)
Inputs
BER publication / Stats SA statistical release P0151; certification month
Outputs
Current index value (I₁) recorded with source reference
Tools
BER subscription portal; Stats SA website; CPA calculation workbook
Standard
JBCC CPA provisions; BER Building Cost Index methodology
  • Work Instruction: Monitor the BER publication schedule — the Building Cost Index is typically released around the 15th of each month for the preceding month
  • Access the BER subscription portal or Stats SA P0151 release and record the index value for the certification month
  • If the contract uses a weighted CPA formula, also retrieve: steel PPI (SIC 2710), fuel PPI, electrical materials PPI from Stats SA
  • Record each index value in the CPA calculation workbook with the publication date and reference number
  • If the current month's index is not yet published at valuation date, use the latest available index and note it for adjustment in the next certificate
  • ✓ Quality Check: Index value matches published source exactly; correct month's index used; source reference documented

  • Work Instruction: Open the CPA baseline record from the contract financial register
  • Confirm the base month index value (I₀) — this was recorded at contract commencement and should never change
  • If using a weighted formula, confirm all component base indices (BER, steel, fuel, electrical)

  • ✓ Quality Check: Base index matches original CPA baseline record; no manual alterations since original recording
TASK 1.4.2-03

Calculate Percentage Index Movement

Compute the CPA adjustment factor by calculating the percentage change between the current index (I₁) and the base index (I₀). This factor represents the escalation percentage to apply to the qualifying value of work.

Role
Quantity Surveyor
SLA
30 minutes
Inputs
Current index I₁ (Task 1.4.2-01); base index I₀ (Task 1.4.2-02)
Outputs
CPA percentage factor
Tools
CPA calculation workbook (formulae pre-loaded)
Standard
JBCC CPA formula: (I₁ − I₀) / I₀
  • Work Instruction: Apply the formula: CPA Factor = (I₁ − I₀) / I₀
  • Example: If I₀ = 152.3 and I₁ = 161.7, then CPA Factor = (161.7 − 152.3) / 152.3 = 0.0617 = 6.17%
  • If using a weighted formula with multiple indices, calculate each component separately and apply the contractual weightings
  • Verify the result is reasonable — compare against previous months' CPA factors for trend consistency
  • A negative factor (deflation) is unusual but possible; apply as a deduction per the same formula
  • ✓ Quality Check: Formula correctly applied; result is consistent with recent trend; weighted components verified if applicable
TASK 1.4.2-04

Determine Qualifying Value from Current Certificate

Extract the qualifying value of work (V) from the current interim payment certificate. The qualifying value excludes materials on site, retention deductions, previous CPA amounts, and any amounts not subject to escalation as specified in the contract.

Role
Quantity Surveyor
SLA
30 minutes
Inputs
Current interim payment certificate (D1.5 valuation output); contract CPA clauses
Outputs
Qualifying amount (V) for CPA calculation
Tools
Payment certificate workbook; CPA calculation template
Standard
JBCC CPA qualifying value exclusions; NEC4 option X1
  • Work Instruction: From the current certificate, take the gross value of work completed to date
  • Deduct the following exclusions per the contract CPA clause: materials on site (not yet incorporated), previous CPA amounts already certified, any non-escalatable items specified in the contract
  • Deduct the qualifying value already used in prior CPA certificates to arrive at the current period's qualifying value
  • For JBCC: V = Current certificate value − exclusions − previous qualifying values
  • Confirm the calculation with reference to the valuation breakdown from D1.5
  • ✓ Quality Check: Exclusions correctly identified per contract; V reconciles to certificate; no double-counting of previous periods
TASK 1.4.2-05

Apply CPA Formula & Calculate Nett Adjustment

Apply the CPA formula to compute the gross CPA amount for the current certificate, then calculate the nett CPA by deducting all previously certified CPA amounts. The nett CPA is the amount added to (or deducted from) the current payment certificate.

Role
Quantity Surveyor
SLA
30 minutes
Inputs
Qualifying value V (Task 1.4.2-04); CPA factor (Task 1.4.2-03); previous CPA certificates
Outputs
Nett CPA amount for current certificate
Tools
CPA calculation workbook
Standard
JBCC CPA formula: CPA = V × (I₁ − I₀) / I₀
  • Work Instruction: Calculate cumulative CPA to date: Cumulative CPA = Total qualifying value to date × CPA Factor
  • Deduct total CPA previously certified: Nett CPA = Cumulative CPA − Sum of all previous CPA certificates
  • Verify: the nett CPA should be positive (escalation) in most months unless there has been deflation or a catch-up adjustment
  • Cross-check: the cumulative CPA as a percentage of the cumulative qualifying value should approximately equal the CPA factor
  • Document the calculation clearly showing: V (current), CPA factor, gross CPA, less previous, nett CPA
  • ✓ Quality Check: Nett CPA reconciles to cumulative less previous; calculation follows contract formula exactly; reasonable when trended
TASK 1.4.2-06

Prepare CPA Calculation Schedule & Append to Certificate

Prepare a formal CPA calculation schedule that shows the full workings transparently. This schedule is appended to the monthly payment certificate as supporting documentation and serves as the audit trail for the CPA amount included.

Role
Quantity Surveyor
SLA
0.5 working day
Inputs
CPA calculation from Task 1.4.2-05; payment certificate
Outputs
CPA calculation schedule; updated payment certificate
Tools
CPA schedule template; payment certificate system
Standard
SACQSP practice guidelines on CPA documentation
  • Work Instruction: Complete the CPA schedule showing: base month and base index (I₀), current month and current index (I₁), CPA factor calculation
  • Show the qualifying value derivation: gross certified value, less exclusions, equals qualifying value (V)
  • Show the CPA amount: V × CPA Factor = Gross CPA; less previous CPA certified = Nett CPA this certificate
  • Include a cumulative summary table showing all CPA amounts certified to date, certificate by certificate
  • Append the CPA schedule to the payment certificate; ensure the nett CPA is correctly reflected as a separate line item in the certificate
  • Issue to the contractor QS for verification before the PA issues the payment certificate
  • ✓ Quality Check: Schedule workings are transparent and verifiable; nett CPA matches certificate line item; contractor QS has reviewed
TASK 1.4.2-07

Record Cumulative CPA in Financial Tracking Register

Update the master financial tracking register with the current month's CPA data. Maintain a running cumulative total that feeds into the anticipated final account calculation and employer cash flow forecast.

Role
Quantity Surveyor
SLA
Immediate (same day as certificate)
Inputs
CPA certificate from Task 1.4.2-06; financial tracking register
Outputs
Updated financial register; revised anticipated final account
Tools
Master financial register; platform reporting module
Standard
SACQSP practice note on contract financial reporting
  • Work Instruction: Update the CPA section of the financial register with: certificate number, certification month, index used, qualifying value, CPA amount, cumulative CPA to date
  • Recalculate the anticipated final account: original contract sum + approved variations + cumulative CPA + projected future CPA
  • Project future CPA exposure based on current escalation trend and remaining contract value — flag if projected final CPA exceeds the original CPA allowance
  • Update the employer's cash flow forecast to reflect the latest CPA trajectory
  • File the CPA calculation schedule in the contract administration filing system under the relevant certificate number
  • ✓ Quality Check: Register is current; cumulative CPA matches sum of all CPA certificates; anticipated final account updated
