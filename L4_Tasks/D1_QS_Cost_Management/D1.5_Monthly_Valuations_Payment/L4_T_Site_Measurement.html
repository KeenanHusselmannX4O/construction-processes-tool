1. Task Overview

This task set expands the Site Measurement & Progress Assessment sub-process into granular work instructions. Each task covers a critical step in the monthly site measurement cycle: reviewing the contractor's claim before the visit, conducting the physical site walk with trade-specific measurement, photographing evidence, performing joint measurements on disputed items, calculating progress percentages per BoQ item, and compiling the measurement records for the valuation calculation. Accurate site measurement per JBCC PBA clause 33 and NEC4 clause 50 is the foundation for fair interim payments.

2. Detailed Task Breakdown

TASK 1.5.1-01

Review Contractor's Monthly Claim Submission

Before the site visit, review the contractor's progress claim submission for the current valuation period. Compare claimed quantities and percentages against the previous month's valuation to identify new claims, significant increases, and potential anomalies requiring focused attention during the site walk.

Role
Quantity Surveyor
SLA
1 working day before site visit
Inputs
Contractor's claim schedule; previous valuation certificate; BoQ with cumulative quantities
Outputs
Pre-visit assessment; flagged items requiring site verification; measurement focus list
Tools
Valuation spreadsheet; platform valuation module; previous certificate data
Standard
JBCC PBA clause 33 (interim payments); monthly cut-off date per contract
  • Work Instruction: Obtain the contractor's claim schedule — ensure it was submitted by the contractual cut-off date (typically 25th of the month)
  • Compare each BoQ line item: contractor's claimed cumulative quantity vs previous month's certified cumulative quantity
  • Calculate the movement (new work claimed) for each item: current claim − previous certified = new claim for this period
  • Flag items where the contractor's claimed progress exceeds 10% increase month-on-month or where new trade sections appear
  • Prepare a measurement focus list prioritising flagged items, high-value trades, and items with historical claim discrepancies
  • ✓ Quality Check: Every BoQ section reviewed; focus list prepared; anomalies documented with specific item references
TASK 1.5.1-02

Conduct Site Walk & Measure Completed Work per Trade

Physically walk the site and measure work completed for each trade section against the BoQ. Apply trade-specific measurement methods: volume calculations for earthworks, dimensional measurement for concrete, wall area measurement for brickwork, roof area from plan, and room-by-room registers for finishes.

Role
QS / Cost Technician
SLA
1–2 working days on site
Inputs
BoQ; previous valuation data; measurement focus list from Task 1.5.1-01; tape measure / laser measure
Outputs
Site measurement notes per trade; field sketches; raw quantity records
Tools
Laser distance meter; measurement book / tablet; site drawings; unit progress tracker
Standard
ASAQS Standard System of Measuring; ±3% accuracy tolerance
  • Work Instruction: Begin with high-value trades first — earthworks (survey data + truck counts), then concrete/structural elements (dimensional measurement vs drawings)
  • For brickwork/blockwork: measure completed wall areas, count completed walls per unit or section
  • For roofing: count completed roofs using plan reference and verify with visual inspection or drone photography
  • For finishes (plaster, paint, tiles): use room-by-room progress register — record percentage complete per room/unit
  • For plumbing/electrical: count installed units against the installation register; verify compliance certificates where applicable
  • Record all measurements with BoQ item references so data maps directly to the valuation schedule
  • ✓ Quality Check: All flagged items from Task 1.5.1-01 physically verified; measurements within ±3% tolerance; field notes legible and referenced
TASK 1.5.1-03

Photograph Key Evidence per BoQ Section

Systematically photograph completed work as visual evidence supporting the measurement records. Photographs must be dated, geo-tagged where possible, and referenced to specific BoQ sections. This evidence supports dispute resolution and audit trail requirements.

Role
QS / Cost Technician
SLA
During site walk (concurrent with Task 1.5.1-02)
Inputs
Camera / smartphone; BoQ section references; site drawing with photo location markers
Outputs
Dated, geo-tagged photographs; photo register linked to BoQ items
Tools
Smartphone with GPS; photo management app; drone (for roofing/large sites)
Standard
SACQSP documentation requirements; project-specific photo protocol
  • Work Instruction: Photograph each major trade section showing the current state of completion — wide shot for context and close-up for detail
  • Ensure photographs are date-stamped and GPS-tagged (enable location services on device)
  • For housing projects: photograph at least one representative unit per block/row showing each stage of completion
  • Include tape measures or scale references in photos of measured items for dimensional verification
  • Label each photograph with the BoQ section reference and brief description immediately after capture
  • Upload photographs to the project document management system within 24 hours of the site visit
  • ✓ Quality Check: Photo register complete with BoQ cross-references; all high-value and disputed items photographed; files uploaded and backed up
TASK 1.5.1-04

Conduct Joint Measurement with Contractor QS on Disputed Items

For BoQ items where the QS assessment differs significantly from the contractor's claim (typically >5% variance), conduct a joint site measurement with the contractor's QS. Both parties measure together and agree on the quantity, which is then binding for the valuation.

Role
QS + Contractor QS
SLA
0.5–1 working day
Inputs
QS measurement (Task 1.5.1-02); contractor's claim; list of items with >5% variance
Outputs
Signed joint measurement records; agreed quantities for disputed items
Tools
Joint measurement template; tape/laser measure; site drawings
Standard
JBCC PBA clause 33; joint measurement best practice per SACQSP
  • Work Instruction: Identify all BoQ items where QS assessment differs from contractor claim by more than 5% — schedule joint measurement
  • Meet the contractor QS on site with both sets of measurement records for comparison
  • Re-measure each disputed item together — both parties use the same measurement points and methodology
  • Record the jointly agreed quantity on the joint measurement template — both parties sign each item
  • Where agreement cannot be reached, record both parties' positions and escalate to the Principal Agent for determination
  • ✓ Quality Check: All disputed items jointly measured; signed by both QS parties; escalated items formally recorded with both positions
TASK 1.5.1-05

Calculate Progress Percentages per BoQ Item

Convert the raw measurement data into progress percentages for each BoQ item and section. Calculate cumulative progress to date, progress for the current period, and compare against the contractor's claimed percentages. This forms the basis for the valuation calculation in D1.5-SP-002.

Role
Quantity Surveyor
SLA
1 working day
Inputs
Site measurement data (Tasks 1.5.1-02 to 1.5.1-04); BoQ total quantities; previous valuation
Outputs
Progress schedule per BoQ item; section summary; variance analysis vs contractor claim
Tools
Valuation spreadsheet; progress calculation template; platform valuation module
Standard
ASAQS measurement rules; cumulative methodology per JBCC
  • Work Instruction: For each BoQ item, calculate: measured quantity to date ÷ total BoQ quantity × 100 = cumulative progress %
  • Calculate this period's progress: current cumulative % − previous month's cumulative % = progress this month
  • Compare QS-assessed progress against contractor's claimed progress for each item — highlight variances >5%
  • For lump-sum items (e.g., prelims time-based), assess progress as a percentage of the activity duration or milestone completion
  • Aggregate item percentages into BoQ section summaries: Substructure %, Superstructure %, Finishes %, etc.
  • Prepare a progress summary table ready for input to the valuation calculation (D1.5-SP-002)
  • ✓ Quality Check: Progress percentages reconcile to measurement data; all BoQ items accounted for; variance analysis complete
TASK 1.5.1-06

Compile Measurement Records for Valuation Input

Consolidate all measurement records, photographs, joint measurement agreements, and progress schedules into a complete valuation input package. This package is the formal hand-off to the valuation calculation sub-process (D1.5-SP-002) and must be audit-ready.

Role
Quantity Surveyor
SLA
1 working day
Inputs
All outputs from Tasks 1.5.1-01 to 1.5.1-05
Outputs
Valuation input package; filed measurement records; hand-off to D1.5-SP-002
Tools
Document management system; valuation file template; platform document store
Standard
SACQSP record-keeping requirements; project audit trail standards
  • Work Instruction: Compile the valuation input package with the following sections: (1) Pre-visit assessment, (2) Site measurement notes, (3) Photographic evidence, (4) Joint measurement records, (5) Progress schedule
  • Ensure all measurement notes are legible, dated, and reference the correct BoQ item numbers
  • Verify that the progress schedule in the package matches the detailed measurement records — no orphan items
  • File the complete package in the project document management system under the current valuation period
  • Formally hand off the progress schedule to the valuation calculation (D1.5-SP-002) with a brief cover note highlighting any unusual items or disputes
  • ✓ Quality Check: Package complete with all 5 sections; cross-references verified; filed and accessible; D1.5-SP-002 notified
